# AI-Powered Telegram Support Bot

🎥 [Watch Demo](https://youtube.com/shorts/zDRrgDAteyc?si=UgCwochNmeiA-Kzb)

An automated Telegram bot for business customer support and lead 
generation, built with a focus on cost efficiency and smart conversation 
management.

## Features
- Bilingual AI responses (Arabic/English) with enforced consistency
- Smart state machine: active → warned → escalated → blocked
- Owner-only admin commands (/leads, /stats, /broadcast, /block, /unblock)
- Automatic lead detection and instant owner notification
- File security scanning (size limits, dangerous extensions)
- Token-optimized AI calls — handles off-topic/repetitive messages 
  without unnecessary API usage
- Auto conversation reset after configurable message limits

## Tech Stack
Python · Flask · SQLite · Telegram Bot API · OpenRouter (LLM fallback chain)

## Status
Core functionality complete and tested. Configuration layer allows 
business-specific customization (hours, FAQ, contact info) without 
touching core logic.
